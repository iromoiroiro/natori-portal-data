# natori-portal

<https://natori-portal.onrender.com/> のデータ

## **本リポジトリのデータを使用する際は、[名取さなコンテンツガイドライン](https://natorisana.com/guideline/) を熟読ください**

## リポジトリ構成

- `search/`
  - [さなちゃんねる字幕検索](https://natori-portal.onrender.com/search) で使用している文字起こしデータ
  - `yyyymmdd.vid.json`
    - 動画 ID ごとの文字起こしデータ
  - `video.json`
    - 動画 ID から動画情報を引くためのデータ

## 使用上の注意

- 本リポジトリのデータを使ったことによるいかなる損害についても、当方は一切の責任を負いかねます
- 本リポジトリが持つデータの正確性は保証しません
- 本リポジトリは適宜更新しますが、常に最新の状態を保つことは保証しません
- データ（内容及び構造）は予告なく変更・削除されることがあります
